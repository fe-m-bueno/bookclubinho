# Bookclubinho

Bookclubinho is a platform for organizing and managing book clubs, enabling readers to create groups, propose readings, track progress, and discuss books together in a collaborative and engaging way.

## Project Structure

| Directory    | Purpose                                                                 |
| ------------ | ----------------------------------------------------------------------- |
| `frontend/`  | Client-side application (UI, components, and user-facing features)      |
| `backend/`   | Server-side API, business logic, and data access layer                  |
| `infra/`     | Infrastructure-as-code, CI/CD pipelines, and deployment configurations  |
| `shared/`    | Shared types, utilities, and constants used across frontend and backend |
| `docs/`      | Project documentation, ADRs, and design references                     |
